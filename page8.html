CSS (Cascading Style Sheets)

Język służący do opisu wyglądu strony.
Dzięki CSS możemy zmieniać sposób wyświetlania elementów strony:

  • Czcionek
  • Odnośników
  • Marginesów
  • Pozycję danego elementu względem okna przeglądarki lub innych elementów

Reguły CSS - struktura

  • Każda reguła CSS zbudowana jest z wartości.
  • Po nazwie własności występuje dwukropek, a cała reguła kończona jest średnikiem.
  • Jeśli nazwa własności składa się z kilku wyrazów, poszczególne wyrazy łączy się krótką kreską(łącznikiem)
  • Jeśli wartość włąsności składa się z kilku członów, odddziela się je spacją.

Selektory

Są to zapisy przypisujące reguły CSS do odpowiednich elementów HTML. Jest wiele sposobób tworzenia selektorów. Najczęściej używane to: element HTML, atrybut id, atrybut class
W nawiasach klamrowych podajemy reguły CSS, które mają być zastosowane do elementów określonych przez selektor. W przykładzie selektor odwołuje się do elementu p. Do wszystkich paragrafów w dokumencie będą zatem zastosowane reguły zawarte w nawiasach klamrowych.

Identyfikatory

Co należy zrobić, jeśli chcemy przypisać reguły CSS tylko do jednego wybranego elementu HTML?
Należy określić atrybut id elementu oraz przypisać reguły CSS za pomocą odpowiednio utworzonego selektora.

Atrybut id jest unikalnym identyfikatorem, będziemy mogli zatem zdefiniować reguły CSS tylko dla tego jednego elementu HTML.

#identyfikator

Selektor poprzedzony znakiem hash (#) będzie odnisił się tylko do elementów strony o tym id.

Co jeśli chcielibyśmy przypisać reguły CSS do wielu różnego typu elemetnów HTML?

Należy określić atrybut class dla takiego elementy oraz przypisać reguły CSS za pomocą odpowiednio utworzonego selektora

Atrybut class można zdefiniować dla każdego elementu HTML

Najlepiej jest korztystać z klas, a nie z id. Przy rozbudowie dokumentu dany element może wystąpić po raz drugi.
